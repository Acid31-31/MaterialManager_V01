TEIL 1: VOR DEM START
─────────────────────

  [ ] 1. Schließe ALLE Explorer-Fenster
        ├─ Besonders: USB_Installation
        ├─ USB_Distribution
        ├─ Installer_Source
        └─ USB_InstallationHelper

  [ ] 2. Öffne Task Manager (Shift + Ctrl + Esc)
        ├─ Suche nach: dotnet
        ├─ Suche nach: msbuild
        ├─ Suche nach: vbcscompiler
        └─ Falls gefunden → Rechtsklick → "Beenden"

  [ ] 3. Restart Computer (optional aber empfohlen)
        └─ Schließe alles & startet neu

TEIL 2: AUTOMATISCHES BEHEBEN (empfohlen!)
────────────────────────────────────────────

  [ ] 1. Gehe zu Ordner:
        C:\Users\hoelz.WIN-G2OC48399EJ\MaterialManager_R03\

  [ ] 2. Suche die Datei:
        AUTOMATISCHES_FEHLERBEHEBUNG.bat

  [ ] 3. Rechtsklick auf die Datei

  [ ] 4. Wähle: "Als Administrator ausführen"

  [ ] 5. Window öffnet sich
        ├─ Viel Text scrollt vorbei (normal!)
        ├─ Lösche alte Build-Dateien
        ├─ Starte Automatisches Build
        └─ Warten Sie!

  [ ] 6. Nach ~15 Minuten:
        └─ USB_InstallationHelper.exe GUI öffnet sich automatisch

  [ ] 7. Im GUI-Fenster:
        ├─ [ ] USB-Stick einstecken (mind. 500 MB frei)
        ├─ [ ] Klick auf "🔄 Aktualisieren" Button
        ├─ [ ] USB in der Liste sehen?
        ├─ [ ] USB-Stick anklicken (auswählen)
        ├─ [ ] Klick auf "💾 Auf USB kopieren" Button
        ├─ [ ] Dialog: "Wirklich kopieren?" → [JA] klicken
        ├─ [ ] Warten auf Progress-Bar (2-10 Min)
        └─ [ ] Message: "✅ Erfolgreich kopiert!" → FERTIG!

TEIL 4: WENN IMMER NOCH FEHLER
────────────────────────────────

  [ ] 1. Öffne PowerShell (als Admin)
        Windows-Taste → Tippe: PowerShell
        Rechtsklick → "Als Admin ausführen"

  [ ] 2. Kopiere diesen Befehl:
        
        cd "C:\Users\hoelz.WIN-G2OC48399EJ\MaterialManager_R03"
        .\COMPLETE_AUTOMATIC_BUILD.ps1 -Verbose

  [ ] 3. Rechtsklick in PowerShell → Paste/Einfügen

  [ ] 4. Enter drücken

  [ ] 5. Warten Sie (~ 15 Min)
        └─ Schaue auf FEHLER/ERROR in rot

  [ ] 6. Screenshot vom Fehler machen
        └─ Windows-Taste + PrintScreen

  [ ] 7. Fehler teilen
        └─ Gib den EXAKTEN Fehlertext an
