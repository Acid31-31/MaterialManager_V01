🔧 TROUBLESHOOTING - WAS WENN...?

Problem 1: USB-Stick wird nicht erkannt
─────────────────────────────────────────

  Lösung:
  1. USB einstecken
  2. Warten 2 Sekunden
  3. Klick auf "🔄 Aktualisieren" Button
  4. USB sollte jetzt sichtbar sein
  
  Wenn immer noch nicht:
  1. USB an einen anderen Port stecken
  2. Nochmal "🔄 Aktualisieren" klicken
  
  Wenn immer noch nicht:
  1. USB-Stick möglicherweise defekt
  2. Versuche mit anderem USB-Stick


Problem 2: Fenster öffnet sich nicht
──────────────────────────────────────

  Lösung:
  1. Prüfe ob .NET 8 installiert ist
     → https://dotnet.microsoft.com/download/dotnet/8.0
  2. Lade "Desktop Runtime" herunter
  3. Installiere
  4. Neustart!
  5. Doppelklick auf EXE nochmal
  
  Wenn immer noch nicht:
  1. Datei: EXE_FEHLER_DIAGNOSE.bat starten
  2. Script zeigt das Problem


Problem 3: Fehler beim Kopieren auf USB
────────────────────────────────────────

  Lösung:
  1. USB-Stick auswerfen (sicher entfernen)
  2. Warten 3 Sekunden
  3. Nochmal einstecken
  4. Nochmal "Auf USB kopieren" versuchen
  
  Wenn immer noch nicht:
  1. Prüfe ob USB-Stick genug Platz hat (mind. 500 MB!)
  2. Lösche alte Dateien von USB
  3. Versuche nochmal


Problem 4: "Installer.exe nicht gefunden"
──────────────────────────────────────────

  Bedeutet: Build wurde noch nicht gemacht!
  
  Lösung:
  1. Fenster schließen
  2. Starte: COMPLETE_AUTOMATIC_BUILD.bat
  3. Warten bis ✅ BUILD ERFOLGREICH
  4. USB_InstallationHelper.exe nochmal starten
  5. Jetzt sollte alles da sein!
