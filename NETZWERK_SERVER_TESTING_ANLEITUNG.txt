📍 LOKALE NETZWERK-FREIGABE ERSTELLEN (Windows 10/11):

METHODE 1: Explorer (Einfach)
────────────────────────────

1. Rechtsklick auf Ordner
2. Freigeben für → Bestimmte Personen
3. Wähle "Alle" oder "Jeder"
4. Berechtigungen: ÄNDERN
5. Freigeben

METHODE 2: PowerShell (Schnell)
──────────────────────────────

PowerShell als Admin:

  New-Item -Path "C:\MaterialManager_TestShare" -ItemType Directory -Force
  
  New-SmbShare -Name "MaterialManager_TestShare" `
    -Path "C:\MaterialManager_TestShare" `
    -FullAccess "Everyone"
  
  # Prüfe:
  Get-SmbShare | grep MaterialManager

🔧 WENN FREIGABE NICHT FUNKTIONIERT:

Problem: "Zugriff verweigert"
Lösung:
  1. Admin-Rechte?
  2. Firewall-Einstellung?
  3. Berechtigungen prüfen:
     Rechtsklick → Sicherheit → Bearbeiten

Problem: "Pfad nicht erreichbar"
Lösung:
  1. Share existiert wirklich?
     net share (in CMD)
  2. Firewall blockiert?
     Windows Defender → Einstellungen → Firewall
     → "Netzwerk-Erkennung" aktivieren
  3. SMB aktivieren:
     Systemsteuerung → Windows-Features
     → "SMB 1.0/CIFS-Dateifreigabe" ✓

Problem: "Lock-Datei funktioniert nicht"
Lösung:
  1. Antivirus blockiert Datei?
  2. Manuelle Lock-Datei löschen:
     C:\MaterialManager_TestShare\.lock (löschen)
  3. Programm neustarten
