🎯 NUTZUNGSSZENARIOS:
═════════════════════════════════════════════════════════════════════════

SZENARIO 1: Schnelle Installation (Recommend für Anfänger)
─────────────────────────────────────────────────────────

1. Doppelklick: START_MENU.bat
2. Wähle: "3) 💾 Installieren"
3. Folge Anweisungen
4. ✅ Fertig in 2-3 Minuten!

→ Installer.exe wird verwendet
→ Graphisches Interface
→ Automatische Konfiguration


SZENARIO 2: Mit Präsentation (Für Präsentationen)
──────────────────────────────────────────────────

1. Doppelklick: START_MENU.bat
2. Wähle: "1) 📺 Präsentation anzeigen"
3. Browser öffnet HTML-Präsentation
4. Nutze Pfeile oder Maus zum Navigieren
5. 12 professionelle Slides

→ PRESENTATION.html wird verwendet
→ Schöne Animationen & Design
→ Ideal für Meetings


SZENARIO 3: Detaillierte Anleitung (Für fortgeschrittene)
──────────────────────────────────────────────────────────

1. Doppelklick: START_MENU.bat
2. Wähle: "2) 📖 Benutzerhandbuch lesen"
3. USER_GUIDE.txt öffnet sich
4. Lese vollständige Dokumentation
5. Folge Schritt-für-Schritt

→ USER_GUIDE.txt wird verwendet
→ Komplette Dokumentation
→ Alle technischen Details


SZENARIO 4: Systemcheck (Vor Installation)
───────────────────────────────────────────

1. Doppelklick: START_MENU.bat
2. Wähle: "5) 🔧 Systemcheck durchführen"
3. Automatische Prüfung läuft
4. Zeigt: OS, RAM, Speicher, .NET, Admin

→ Prüft Systemvoraussetzungen
→ Zeigt potenzielle Probleme
→ Empfehlungen anzeigen

🚀 INSTALLATION SCHRITTE:
═════════════════════════════════════════════════════════════════════════

Option A: Mit Installer.exe
──────────────────────────

1. START_MENU.bat → "Installieren"
2. Installer.exe startet
3. Graphisches Menü folgen
4. Zielverzeichnis auswählen
5. Installation lädt (~2-3 Min)
6. Desktop-Shortcut erstellt
7. ✅ Fertig!

→ Einfach & automatisch
→ Empfohlen für die meisten


Option B: Direktes Programm starten
───────────────────────────────────

1. Explorer öffnen
2. Zu Programm/ navigieren
3. MaterialManager_R03.exe doppelklicken
4. ✅ Programm startet sofort!

→ Keine Installation nötig
→ Portable Version
→ Für USB-Betrieb


🔐 LIZENZAKTIVIERUNG:
═════════════════════════════════════════════════════════════════════════

Nach Installation:

1. Programm öffnet Lizenzfenster
2. Gib deinen Lizenzschlüssel ein
3. Klick "Aktivieren"
4. Internetverbindung prüft Lizenz
5. ✅ Aktiviert!

Dein Lizenzschlüssel:
  • Wurde per Email zugesendet
  • Oder liegt diesem Packet bei
  • Prüfe: LICENSE.txt


🌐 NETZWERK-BETRIEB:
═════════════════════════════════════════════════════════════════════════

Für mehrere Benutzer:

1. Gemeinsamen Ordner erstellen
2. Im Programm: Einstellungen → Netzwerk
3. Freigabepfad eingeben
4. Speichern
5. Andere Benutzer tun dasselbe
6. ✅ Automatische Synchronisierung!

Siehe: Anleitung/NETZWERK_SETUP.md
