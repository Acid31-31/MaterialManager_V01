3. INSTALLATION GUIDE
═══════════════════════════════════════════════════════════════════════════

Step-by-Step Installation:
──────────────────────────

STEP 1: Insert USB Drive
  1. Connect the USB stick to your computer
  2. Wait for Windows to recognize it
  3. Note the drive letter (e.g., D:\)

STEP 2: Run the Installer
  1. Open File Explorer
  2. Navigate to USB: MaterialManager_R03_Installer.exe
  3. Double-click the installer
  4. Windows may show a security warning → Click "Run anyway"

STEP 3: Installation Process
  1. Select installation language (Deutsch / English)
  2. Accept license terms
  3. Choose installation path (default: C:\Program Files\)
  4. Click "Install"
  5. Wait for installation to complete (~2-3 minutes)

STEP 4: Complete Installation
  1. Click "Finish"
  2. Program starts automatically
  3. Create admin user account
  4. Enter license key (provided separately)
  5. Installation complete ✅

Default Installation Path:
  C:\Program Files\MaterialManager_R03\

5. NETWORK CONFIGURATION
═══════════════════════════════════════════════════════════════════════════

Multi-User Setup:
─────────────────

MaterialManager supports multiple users working simultaneously. 
This requires a shared network folder.

Configuration Steps:

STEP 1: Create Shared Folder
  1. Create folder: C:\MaterialManager_Share\
  2. Right-click → Properties → Share
  3. Set permissions: Everyone = Full Access
  4. Note the share path: \\localhost\MaterialManager_Share\

STEP 2: Configure in MaterialManager
  1. In program: Settings → Network
  2. Enable network mode ✓
  3. Enter share path
  4. Click "Test Connection"
  5. Should show "Connection successful"

STEP 3: Multi-User Testing
  1. Start program on first computer
  2. Start program on second computer
  3. Both connect to same network folder
  4. Changes are synchronized in real-time

Network Requirements:
  • Stable LAN connection
  • SMB file sharing enabled
  • All computers on same network
  • Sufficient disk space on server (1+ GB)

7. TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

Common Issues & Solutions:
──────────────────────────

ISSUE: Installation fails with "Access Denied"
──────────────────────────────────────────────
SOLUTION:
  1. Run installer as Administrator
  2. Right-click → "Run as Administrator"
  3. Disable antivirus temporarily
  4. Ensure 500 MB free disk space
  5. Retry installation

ISSUE: Program crashes on startup
─────────────────────────────────
SOLUTION:
  1. Uninstall completely
  2. Delete C:\Users\[User]\AppData\Local\MaterialManager_R03\
  3. Restart computer
  4. Reinstall from USB

ISSUE: Network connection not working
────────────────────────────────────
SOLUTION:
  1. Verify network folder exists and is shared
  2. Test path manually: Start → Run → \\localhost\MaterialManager_Share\
  3. Check permissions: Everyone = Change
  4. Disable Windows Firewall temporarily
  5. Verify SMB is enabled (Windows Features)
  6. Restart program

ISSUE: File is locked / Cannot save
───────────────────────────────────
SOLUTION:
  1. Ensure only one user is editing material
  2. Check lock file exists: .lock
  3. Delete .lock file if stuck
  4. Restart program
  5. Try again

ISSUE: License key not accepted
───────────────────────────────
SOLUTION:
  1. Verify key is correct (check for typos)
  2. Ensure internet connection is active
  3. Check license hasn't expired
  4. Try re-entering key without spaces
  5. Contact support with license details
