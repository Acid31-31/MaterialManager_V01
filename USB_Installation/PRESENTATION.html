Was ist MaterialManager R03?

MaterialManager R03 ist eine professionelle Software zur Verwaltung von Materialbeständen in Blechlägern.

Hauptmerkmale:

  • ✅ Bestandsverwaltung in Echtzeit
  • ✅ Netzwerk-fähig (mehrere PCs)
  • ✅ Automatische Datensynch
  • ✅ Benutzerlizenzen
  • ✅ Excel-Export
  • ✅ Audit-Logging

Systemvoraussetzungen

🖥️ Hardware:

  • Prozessor: Intel/AMD ab Pentium 4 oder gleichwertig
  • RAM: Minimum 2 GB (empfohlen 4 GB+)
  • Festplatte: 500 MB freier Speicher

💻 Software:

  • Betriebssystem: Windows 10 oder Windows 11 (64-bit)
  • .NET 8 Runtime (wird mitgeliefert)
  • Excel 2016+ (optional, für Export)

Hauptfunktionen

📊 Bestandsverwaltung

Verwaltung aller Materialarten mit automatischer Bestandsführung

🔗 Netzwerk-Betrieb

Mehrere Benutzer gleichzeitig, zentraler Datenbestand

💾 Datensicherung

Automatische Backups und Versionsverwaltung

📈 Berichte

Detaillierte Statistiken und Excel-Export

🔐 Benutzerrechte

Granulare Berechtigungen pro Benutzer

📝 Audit-Log

Komplette Änderungshistorie aller Transaktionen

Installation

Schritt-für-Schritt:

1. USB-Stick einstecken

Schließe den USB-Stick an Ihren Computer an.

2. Installer starten

Doppelklick auf MaterialManager_R03_Installer.exe

3. Installation durchführen

Folge den Anweisungen des Installers (ca. 2-3 Minuten)

4. Programm starten

Desktop-Shortcut oder Start-Menü verwenden

5. Lizenzen aktivieren

Gib Deinen Lizenzschlüssel bei der ersten Nutzung ein

Netzwerk-Konfiguration

Für mehrere Benutzer:

Wichtig: MaterialManager kann im Netzwerk für mehrere Benutzer gleichzeitig verwendet werden!

Setup-Schritte:

  1. Gemeinsamen Netzwerk-Ordner erstellen
  2. Berechtigungen für alle Benutzer setzen
  3. In MaterialManager: Einstellungen → Netzwerk
  4. Pfad eingeben: \\Server\MaterialManager\
  5. Speichern und fertig!

Erste Schritte

Nach der Installation:

  1. Material-Katalog einrichten

    Definiere die Materialarten (Bleche, Profile, etc.)

  2. Lagerplätze konfigurieren

    Richte deine Lagerstruktur ein

  3. Benutzer erstellen

    Füge Benutzer mit entsprechenden Rechten hinzu

  4. Bestand erfassen

    Starte die erste Bestandsaufnahme

  5. Schulung abschließen

    Trainiere Dein Team in der Nutzung

Häufige Probleme & Lösungen

Problem: Installation schlägt fehl

  • Prüfe, ob 500 MB Festplattenspeicher frei ist
  • Führe als Administrator aus
  • Deaktiviere temporär Antivirus-Software

Problem: Netzwerk nicht erreichbar

  • Prüfe Netzwerk-Pfad in Einstellungen
  • Firewall-Ausnahme aktivieren
  • Berechtigungen prüfen

Problem: Lizenzierung funktioniert nicht

  • Prüfe Lizenzschlüssel auf Tippfehler
  • Internetverbindung prüfen
  • Kontaktiere Support

Best Practices

Für optimale Nutzung:

  • ✅ Regelmäßige Backups: Täglich automatische Sicherung
  • ✅ Benutzertraining: Schule dein Team regelmäßig
  • ✅ Netzwerk-Stabilität: Verlässliche Verbindung ist wichtig
  • ✅ Datenbereinigung: Archiviere alte Datensätze monatlich
  • ✅ Audit-Logs prüfen: Wöchentliche Überprüfung für Kontrolle

💡 Tipp: Verwende die automatischen Berichte für regelmäßige Kontrollen!
