# 📊 MaterialManager R03 - Präsentations-Version

Hallo zusammen,

anbei erhaltet ihr die **MaterialManager R03 Presentations-Version** - eine moderne Lagerverwaltung für Blechlager!

## ✨ FEATURES

### 🏢 **Materialverwaltung**
- ✅ Bestandsverwaltung mit Gewicht-Tracking
- ✅ Regalauslastung in Echtzeit
- ✅ Schnelle Suche & Filter (Material, Legierung, Form)
- ✅ EU-Palette & Rest-Material Tracking
- ✅ Reservierungs-System für Material

### 🔄 **Netzwerk & Synchronisation**
- ✅ Multi-PC Auto-Sync (Live-Updates)
- ✅ Netzwerk-freigabe Support
- ✅ Automatische Datensicherung
- ✅ Update-Prüfung via GitHub

### 📊 **Analysen & Reports**
- ✅ Verschnitt-Analyse (Abfallquoten)
- ✅ Verbrauchsstatistik
- ✅ Niedrige Bestände Warnung
- ✅ Rückgängig/Wiederherstellen (Undo/Redo)

### 💾 **Export & Datenmanagement**
- ✅ CSV Export (Excel-kompatibel)
- ✅ XML Export (ERP-Systeme)
- ✅ JSON API (Zukunftssicher)
- ✅ Automatische Backups

## 🚀 INSTALLATION (3 Schritte)

### Schritt 1: Datei entpacken
```
MaterialManager_V01_Release.zip → Entpacken
```

### Schritt 2: Installer ausführen
```
Doppelklick auf: Install.bat
```

### Schritt 3: Starten
```
Menü → MaterialManager R03 starten
```

**Fertig! Keine Admin-Rechte nötig. Funktioniert lokal + Netzwerk!**

## ❓ HÄUFIGE FRAGEN

**F: Brauche ich einen Server?**
A: Nein! Die App läuft lokal. Netzwerk ist optional.

**F: Können mehrere PCs die gleichen Daten benutzen?**
A: Ja! Via Netzwerk-Sync (siehe NETZWERK_ANLEITUNG.txt)

**F: Wie importiere ich alte Daten?**
A: Über CSV-Import in der App (Menü → Datei → Import)

**F: Was kostet das?**
A: Kostenlos! Open-Source.
