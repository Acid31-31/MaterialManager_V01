## ✨ Features

### 📦 Intelligentes Regal-System
- **10 spezialisierte Regale** (A-J)
- **Automatische Zuordnung** nach Material und Größe
- **EU Paletten** für Reste und kleine Materialien

### 🏷️ Restnummer-System
- **Format**: KW_TAG-NR (z.B. 50_DI-01)
- **Automatische Generierung**
- **Täglicher Reset**

### 📊 Material-Verwaltung
- Excel Import/Export
- Suchfunktion
- Material-Filterung (GF, MF, KF, Rest)
- Gewichtsberechnung

### 🎨 Modern UI
- Dark Theme Design
- Logo-Integration
- Intuitive Bedienung
- Regal-Auslastungsanzeige

### 📚 Regal-Logik

#### Regal A, F, J
- Mittelformat (MF)
- 800 kg Kapazität pro Regal
- Automatische Last-Verteilung

#### Regal B
- Aluminium/Edelstahl ≥6mm
- 1.500 kg Kapazität

#### Regal C
- Hochfeste Stähle (S355, S460, HB400, HB500)
- 1.000 kg Kapazität

#### Regal D
- Baustähle (S235)
- 1.000 kg Kapazität

#### Regal E
- Sonstige Großformate
- 1.000 kg Kapazität

#### Regal G, H, I
- Kleinformat (KF)
- 600 kg Kapazität pro Regal
- Automatische Last-Verteilung

#### EU Palette
- Reste ≤ 1500x1300mm
- Kleine Materialien
- 2.000 kg Kapazität

## 🚀 Installation

1. Entpacken Sie den Ordner
2. Starten Sie **MaterialManager_V01.exe**
3. Keine zusätzliche Installation erforderlich!

## 📖 Bedienung

### Material hinzufügen
1. Klicken Sie auf **"Material neu"**
2. Füllen Sie die Felder aus
3. Lagerort wird automatisch bestimmt

### Material bearbeiten
- **Doppelklick** auf eine Zeile in der Tabelle

### Material löschen
1. Wählen Sie eine Zeile aus
2. Klicken Sie auf **"Material löschen"**

### Suchen
- Geben Sie Text in die Suchleiste ein
- Sucht in: Material, Legierung, Oberfläche, Maß, Restnummer

### Filtern
- Wählen Sie ein Format: **Alle**, **GF**, **MF**, **KF**, **Rest**

### Excel Export
1. Klicken Sie auf **"Export Excel"**
2. Wählen Sie Speicherort
3. Excel-Datei wird erstellt

### Excel Import
1. Klicken Sie auf **"Import Excel"**
2. Wählen Sie Excel-Datei
3. Materialien werden importiert
