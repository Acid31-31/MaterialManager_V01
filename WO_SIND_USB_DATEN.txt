📈 PROZESS - WIE WERDEN DATEN HIER HINGESTECKT?

Schritt 1: MASTER_SETUP ausführen
─────────────────────────────────
  .\MASTER_SETUP.ps1 -Action Full
  
  Ergebnis:

  └─ Installer.exe wird kopiert → USB_Installation\Installer.exe


Schritt 2: SETUP_PROGRAMM.bat ausführen
────────────────────────────────────────
  cd USB_Installation
  SETUP_PROGRAMM.bat
  
  Ergebnis:
  ├─ MaterialManager_R03.exe wird kopiert
  │  Quelle: ..\USB_Distribution\MaterialManager_R03.exe
  │  Ziel: USB_Installation\Programm\
  │
  └─ *.dll werden kopiert (rekursiv)
     Quelle: ..\USB_Distribution\*
     Ziel: USB_Installation\Programm\


Schritt 3: USB-Stick vorbereiten
──────────────────────────────────
  copy-item -Path "USB_Installation\*" `
            -Destination "D:\" -Recurse
  
  Ergebnis:

  ├─ Installer.exe               (170 MB)
  ├─ Programm/
  │  ├─ MaterialManager_R03.exe
  │  └─ *.dll                    (150 MB)

  └─ (weitere Dateien)

Schritt 4: Kunde erhält USB
────────────────────────────
  Kunde:
  1. USB einstecken
  2. Installer.exe doppelklicken
  3. Admin-Abfrage bestätigen
  4. Installation läuft ✓


Schritt 5: Nach 30 Tagen - Lizenz
──────────────────────────────────
  Kunde:
  1. Hardware-ID kopieren (Menü → Lizenzinformationen)
  2. Email an dich senden

  1. GENERATE_LICENSE.bat ausführen
  2. Hardware-ID eingeben
  3. Kundenname eingeben
  4. Lizenzschlüssel erhalten
  5. Zurück an Kunde senden
  
  Kunde:
  1. Lizenzschlüssel eingeben (Menü → Lizenzierung)
  2. Vollversion läuft ✓
