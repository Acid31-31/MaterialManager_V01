🔄 KOMPLETTER WORKFLOW:

  1. Installer bauen:
     .\Installer_Source\Build-Installer.ps1 -Action Package

  2. Zu USB_Installation kopieren:
     copy bin\Release\net8.0-windows\win-x64\publish\Installer.exe `
          USB_Installation\

  3. Programm-Dateien füllen:
     cd USB_Installation
     SETUP_PROGRAMM.bat

  4. Auf USB-Stick kopieren:
     copy-item -Path "USB_Installation\*" -Destination "D:\" -Recurse

  5. Auf Zielrechner:
     USB einstecken
     Installer.exe doppelklicken
     Admin-Abfrage bestätigen
     ✅ Installation läuft

🆘 WENN ANTIVIRUS IMMER NOCH WARNT:

  Grund: Neue EXE noch nicht vertrauenswürdig

  Lösungen:

  1. WHITELIST:
     - Admin öffnet Antivirus
     - Installer.exe zur Whitelist hinzufügen
     - Problem erledigt

  2. CODE-SIGNING:
     - Kaufe Zertifikat (DigiCert, Sectigo, ~100€/Jahr)
     - Signiere EXE damit
     - Windows erkennt Publisher automatisch
     - Keine Warnungen mehr

  3. SMARTER SCREEN (wenn Windows warnt):
     - "Weitere Informationen" klicken
     - "Trotzdem ausführen" klicken
     - Wird dann vertraut (cached)
