3. USB-INSTALLATION
═══════════════════════════════════════════════════════════════════════════════

3.1 USB-PAKET ERSTELLEN
──────────────────────────────────────────────────────────────────────────────

1. Führe aus: Build_USB_Vollversion.bat

   Das Script:
   ✓ Kompiliert Vollversion (Self-Contained)
   ✓ Erstellt USB-Paket-Struktur
   ✓ Kopiert alle erforderlichen Dateien
   ✓ Fügt Installer und Dokumentation hinzu

2. Ergebnis: Ordner "USB_Package" mit:
   
   USB_Package/
   ├── INSTALL_VOLLVERSION.bat      ← Installations-Script
   ├── README_VOLLVERSION.txt       ← Anleitung für Kunden
   ├── LICENSE.txt                  ← Lizenzbestimmungen
   ├── COPYRIGHT.txt                ← Urheberrechtshinweis
   ├── VERSION_INFO.txt             ← Build-Informationen
   ├── autorun.inf                  ← Auto-Start bei USB-Einstecken
   ├── LIZENZ_GENERATOR_INTERN.bat  ← NUR FÜR DICH! (geheim halten)
   └── App/                         ← Alle Programmdateien
       ├── MaterialManager_R03.exe
       ├── *.dll (alle Dependencies)
       └── Assets/

3. Kopiere "USB_Package" auf USB-Stick (min. 500 MB)


3.2 USB-STICK ÜBERGABE
──────────────────────────────────────────────────────────────────────────────

✓ Vorbereitung:
  • USB-Stick mit "MaterialManager R03 - Vollversion" beschriften
  • Optional: Aufkleber mit Logo und Kontaktdaten
  • WICHTIG: LIZENZ_GENERATOR_INTERN.bat VOR Übergabe löschen!

✓ Übergabe-Dokumente:
  • USB-Stick mit Installation
  • Gedruckte Anleitung (README_VOLLVERSION.txt)
  • Rechnung / Lieferschein
  • Optional: Visitenkarte mit Support-Kontakt


3.3 INSTALLATION BEIM KUNDEN
──────────────────────────────────────────────────────────────────────────────

ANLEITUNG FÜR KUNDEN:

1. USB-Stick einstecken
2. Rechtsklick auf "INSTALL_VOLLVERSION.bat"
3. "Als Administrator ausführen"
4. Folge den Anweisungen auf dem Bildschirm
5. Programm wird installiert nach: C:\Program Files\MaterialManager_R03
6. Desktop-Verknüpfung wird erstellt

WICHTIG:
• Admin-Rechte erforderlich
• Antivirus kann Warnung zeigen (ist normal bei Self-Contained Apps)
• Windows SmartScreen: "Trotzdem ausführen" klicken

4.2 LIZENZSCHLÜSSEL GENERIEREN (NUR FÜR DICH!)
──────────────────────────────────────────────────────────────────────────────

WICHTIG: Lizenz-Generator NIEMALS an Kunden weitergeben!

OPTION A: Mit Lizenz-Generator-Script
──────────────────────────────────────────────────────────────────────────────
1. Öffne LIZENZ_GENERATOR_INTERN.bat (aus deinem Backup, NICHT vom USB!)
2. Gebe Hardware-ID des Kunden ein
3. Wähle Lizenztyp:
   1 = Einzelplatz (499 EUR)
   2 = Mehrplatz 3 (1.199 EUR)
   3 = Firmenlizenz 10 (2.990 EUR)
   4 = Enterprise
4. Lizenzschlüssel wird generiert (z.B. "S1A3-F7B9-D2E1-C8F4")
5. Kopiere Lizenzschlüssel


OPTION B: Manuell im Programm
──────────────────────────────────────────────────────────────────────────────
1. Starte MaterialManager_R03.exe mit Parameter:
   MaterialManager_R03.exe --generate-license "HARDWARE_ID" 1
   
   Parameter 1 = Einzelplatz
   Parameter 2 = Mehrplatz 3
   Parameter 3 = Firmenlizenz 10
   Parameter 4 = Enterprise

2. Lizenzschlüssel wird in Console angezeigt


WICHTIG: Dokumentiere alle vergebenen Lizenzen!
──────────────────────────────────────────────────────────────────────────────
Lege eine Excel-Tabelle an:

Datum      | Kunde           | Hardware-ID      | Lizenzschlüssel   | Typ
