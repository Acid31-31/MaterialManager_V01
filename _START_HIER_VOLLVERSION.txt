SCHRITT 2: PROJEKT KOMPILIEREN
----------------------------------------------------------------------------

1. Öffne Visual Studio
2. Öffne MaterialManager_V01.csproj
3. Stelle sicher, dass alle Packages installiert sind:
   - ClosedXML
   - System.Management (NEU!)
4. Build > Rebuild Solution
5. Teste das Programm:
   - Starte MaterialManager_V01.exe
   - Prüfe Lizenz-Dialog: Hilfe > Lizenz aktivieren
   - Prüfe Copyright-Hinweis
   - Prüfe Demo-Version Warnung


SCHRITT 3: USB-PAKET ERSTELLEN
----------------------------------------------------------------------------

1. Führe aus: Build_USB_Vollversion.bat
   (WICHTIG: Als Administrator ausführen!)

2. Das Script erstellt automatisch:
   USB_Package/
   +-- INSTALL_VOLLVERSION.bat
   +-- README_VOLLVERSION.txt
   +-- LICENSE.txt
   +-- COPYRIGHT.txt
   +-- VERSION_INFO.txt
   +-- autorun.inf
   +-- LIZENZ_GENERATOR_INTERN.bat    ? NUR FÜR DICH!
   +-- App/
       +-- (alle Programmdateien)

3. Prüfe das Paket:
   - Größe ca. 200-500 MB
   - Alle Dateien vorhanden
   - App/MaterialManager_V01.exe existiert


SCHRITT 4: FÜR KUNDENÜBERGABE VORBEREITEN
----------------------------------------------------------------------------

1. WICHTIG: Lösche LIZENZ_GENERATOR_INTERN.bat aus USB_Package!
   (Dieser ist NUR für dich, niemals an Kunden!)

2. Kopiere USB_Package auf USB-Stick (min. 1 GB)

3. Beschrifte USB-Stick:
   "MaterialManager V01 - Vollversion"

4. Optional: Erstelle Aufkleber mit Logo + Kontaktdaten

SCHRITT 6: LIZENZIERUNG (nach Verkauf)
----------------------------------------------------------------------------

1. KUNDE INSTALLIERT:
   +-- Kunde führt INSTALL_VOLLVERSION.bat aus
   +-- Programm wird installiert

2. KUNDE SENDET HARDWARE-ID:
   +-- Kunde startet Programm
   +-- Hilfe > Lizenz aktivieren
   +-- Kunde kopiert Hardware-ID
   +-- Kunde sendet dir Hardware-ID per E-Mail

3. DU GENERIERST LIZENZSCHLÜSSEL:
   +-- Sichere Kopie: LIZENZ_GENERATOR_INTERN.bat (nur in deinem Backup!)
   +-- Führe Generator aus
   +-- Hardware-ID eingeben
   +-- Lizenztyp wählen (1-4)
   +-- Lizenzschlüssel wird generiert

4. DU SENDEST LIZENZSCHLÜSSEL:
   +-- E-Mail an Kunden mit Lizenzschlüssel
   +-- Anleitung zur Aktivierung
   +-- Willkommens-Nachricht

5. KUNDE AKTIVIERT:
   +-- Hilfe > Lizenz aktivieren
   +-- Lizenzschlüssel eingeben
   +-- Name/Firma eingeben
   +-- Aktivieren - FERTIG!

6. DOKUMENTIEREN:
   +-- Lizenz-Vergabe in Excel-Tabelle festhalten!
