👥 SCHRITT 3: INSTALLATION AUF MEHREREN PCs

Für JEDEN Kunden-PC:

1. USB-Stick einstecken (z.B. D:\)

2. Explorer öffnen
   → D:\MaterialManager_R03_Installer.exe

3. Doppelklick auf: MaterialManager_R03_Installer.exe
   ⏳ Installation läuft automatisch (~2-5 Minuten)

4. Windows fragt nach Admin-Bestätigung
   → "Ja" klicken

5. Installation-Fenster öffnet sich
   → "Weiter" klicken
   → "Installieren" klicken
   → Wartet...
   → "Fertig" klicken

6. ✅ FERTIG!
   Programm ist INSTALLIERT!
   Desktop-Shortcut wurde erstellt!

7. Doppelklick auf: MaterialManager R03 (Desktop)
   → Programm STARTET!

⚙️  KONFIGURATION NACH INSTALLATION

Nach Installation können Kunden konfigurieren:

1. Datenbank-Verbindung
   → Tools → Netzwerk Einstellungen
   → SQL Server oder SQLite auswählen

2. Benutzereinstellungen
   → Edit → Einstellungen
   → Sprache, Design, etc.

3. Lizenzen
   → Help → Lizenzverwaltung
   → Lizenzschlüssel eingeben/ändern

🚨 FEHLERBEHANDLUNG

Problem: "Admin-Rechte erforderlich"
Lösung: Rechtsklick → "Als Administrator ausführen"

Problem: "Installation fehlgeschlagen"
Lösung: 
  • Prüfe Festplattenspeicher (mind. 500 MB frei)
  • Schließe das Programm wenn es läuft
  • Versuche erneut

Problem: "Datenbank-Fehler beim Start"
Lösung:
  • Tool GENERATE_LICENSE.bat starten
  • Lizenzschlüssel aktualisieren
